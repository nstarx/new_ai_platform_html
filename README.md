# new_ai_platform_html

Static HTML dashboard UI for the NStarX AI platform with AI Zones and deployments.

## Overview

A static HTML/CSS/JS dashboard providing a web interface for managing AI Zones and deployments on the NStarX AI platform.

## Pages

- `index.html` - Main dashboard
- `ai-zones.html` - AI Zones management
- `deployments.html` - Deployment management
- `login.html` / `register.html` - Authentication pages
- `profile.html` - User profile
- `table.html` - Data table view
- `assets/` - CSS, JS, and image assets

## Usage

Open `index.html` in a browser or serve with any static file server.
